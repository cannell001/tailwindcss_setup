# tailwindcss_setup

Each Tailwind setup is on its own branch:

| Method  | Branch | Description |
|----------|--------|-------------|
| CDN      | `cdn` | Use the Tailwind CDN script directly in HTML |
| CLI      | `cli` | Build CSS using Tailwind CLI |
| PostCSS  | `postcss` | Tailwind + autoprefixer via PostCSS |
| Vite     | `vite` | Tailwind with Vite dev server |

## Switch branches
```bash
git checkout <branch>
